<span data-ttu-id="b7188-117">如果您有幾個邏輯部分所組成的複雜頁面時，可能很有用，才能使用它自己的部分檢視為每個片段。</span><span class="sxs-lookup"><span data-stu-id="b7188-117">If you have a complex page made up of several logical pieces, it can be helpful to work with each piece as its own partial view.</span></span> <span data-ttu-id="b7188-118">每個頁面片段就可以在獨立於其餘的頁面上，檢視，因為它只包含整體頁面結構和呼叫來呈現部分檢視網頁本身的檢視會變得簡單許多。</span><span class="sxs-lookup"><span data-stu-id="b7188-118">Each piece of the page can be viewed in isolation from the rest of the page, and the view for the page itself becomes much simpler since it only contains the overall page structure and calls to render the partial views.</span></span>

<span data-ttu-id="b7188-119">提示： 請依照下列[不重複自行原則](http://deviq.com/don-t-repeat-yourself/)在檢視中。</span><span class="sxs-lookup"><span data-stu-id="b7188-119">Tip: Follow the [Don't Repeat Yourself Principle](http://deviq.com/don-t-repeat-yourself/) in your views.</span></span>

### <a name="partial-view-discovery"></a><span data-ttu-id="b7188-138">部分檢視探索</span><span class="sxs-lookup"><span data-stu-id="b7188-138">Partial View Discovery</span></span>

<span data-ttu-id="b7188-139">當參考的部分檢視，您可以參考其位置數種方式：</span><span class="sxs-lookup"><span data-stu-id="b7188-139">When referencing a partial view, you can refer to its location in several ways:</span></span>

```text
// Uses a view in current folder with this name
// If none is found, searches the Shared folder
@Html.Partial("ViewName")

// A view with this name must be in the same folder
@Html.Partial("ViewName.cshtml")

// Locate the view based on the application root
// Paths that start with "/" or "~/" refer to the application root
@Html.Partial("~/Views/Folder/ViewName.cshtml")
@Html.Partial("/Views/Folder/ViewName.cshtml")

// Locate the view using relative paths
@Html.Partial("../Account/LoginPartial.cshtml")
```

<span data-ttu-id="b7188-140">您可以在不同檢視資料夾具有相同名稱的不同部分檢視。</span><span class="sxs-lookup"><span data-stu-id="b7188-140">You can have different partial views with the same name in different view folders.</span></span> <span data-ttu-id="b7188-141">當參考檢視的名稱 （不含副檔名），每個資料夾中的檢視會使用部分檢視它們的相同資料夾中。</span><span class="sxs-lookup"><span data-stu-id="b7188-141">When referencing the views by name (without file extension), views in each folder will use the partial view in the same folder with them.</span></span> <span data-ttu-id="b7188-142">您也可以指定預設部分檢視，若要使用，放在*共用*資料夾。</span><span class="sxs-lookup"><span data-stu-id="b7188-142">You can also specify a default partial view to use, placing it in the *Shared* folder.</span></span> <span data-ttu-id="b7188-143">共用的部分檢視將供沒有自己的版本部分檢視的任何檢視。</span><span class="sxs-lookup"><span data-stu-id="b7188-143">The shared partial view will be used by any views that don't have their own version of the partial view.</span></span> <span data-ttu-id="b7188-144">您可以有預設的部分檢視 (在*共用*)，這會覆寫具有相同名稱做為父檢視的相同資料夾中的部分檢視。</span><span class="sxs-lookup"><span data-stu-id="b7188-144">You can have a default partial view (in *Shared*), which is overridden by a partial view with the same name in the same folder as the parent view.</span></span>
